An Android/Raspberry Pi/OUYA development platform based around TinyScheme.
==========================================================================

Includes:
* Ability to build android UI entirely via scheme scripts
* Access to loads of android functionality (wifi, network, sound, GPS, camera etc)
* Full hardware accelerated 3D engine (based on fluxus)
* Sound synthesis library
* Jellyfish: an experimental bytecode compiler for procedural graphics
* Native Raspberry Pi GPU support
* OUYA controller support
